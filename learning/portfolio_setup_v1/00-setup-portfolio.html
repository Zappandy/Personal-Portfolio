How I Built This Portfolio Site

Learning Log · v1

This document tracks the key steps I took to set up this site using GitHub Pages and plain HTML.

Step-by-step summary

  1. Created a new GitHub repository named Personal-Portfolio.
  2. Added a basic index.html file with my portfolio content.
  3. Pushed the code to GitHub using Git:
git init
git remote add origin https://github.com/Zappandy/Personal-Portfolio.git
git add .
git commit -m "Initial commit"
git push -u origin main
  1. Enabled GitHub Pages under Settings > Pages.
  2. Confirmed the site is live at Zappandy.github.io/Personal-Portfolio.
  3. Created a /learning/ directory to document the journey.
