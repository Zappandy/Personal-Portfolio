# Image assets

Naming convention. Drop files here and the pre-wired `<img>` tags will pick them up.

## Rules
- Disclosure test: "Could this image *only* have come from inside the company?" If yes, don't use it.
- Prefer self-authored diagrams (Excalidraw / Mermaid) and already-public assets (published figures, official photos).
- Keep one consistent visual style across all images. Match the accent to the site blue `#0071e3`.
- Recommended size: ~1200px wide, transparent or white background. PNG or SVG.
